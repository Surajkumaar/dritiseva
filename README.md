# Dritiseva - Elder Care & Nursing Services

Premium home care and patient care platform providing trained nurses and caregivers for seniors and bedridden patients at home.

## Tech Stack
- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS + Shadcn UI
- **Routing**: React Router Dom
- **Icons**: Lucide React
- **Icons**: Google Fonts (Inter, Georgia)

## Scripts
- `npm run dev`: Start local development server
- `npm run build`: Build for production
- `npm run preview`: Preview the production build locally
